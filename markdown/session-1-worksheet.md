# 📝 ใบงานประจำคาบที่ 1: Build & Deploy Your First Webpage with AI

**เป้าหมาย:** 
1. สร้างไฟล์ `requirements.md` เพื่อวางแผนโครงสร้างเว็บ
2. ใช้ AI (Antigravity/Claude) ในการเขียนโค้ด `index.html` จาก Requirement
3. นำเว็บไซต์ขึ้นออนไลน์สู่สายตาชาวโลกด้วย **GitHub Pages** ผ่านคำสั่ง **Git CLI**

---

## 🟢 Part 1: ภารกิจในห้องเรียน (In-Class Mission)
*กรุณาทำภารกิจต่อไปนี้ให้เสร็จและส่งลิงก์ก่อนหมดเวลาเรียน*

### Step 1: ให้ AI เป็น Copywriter & Artist (สร้างเนื้อหาและรูปภาพ)
ก่อนจะสร้างเว็บ เราต้องมี "วัตถุดิบ" ก่อน! เราจะใช้ AI ช่วยคิดคอนเทนต์และวาดรูปให้ โดยใช้เทคนิคการคุยให้เชื่อมโยงกับ Markdown

1. **สร้างเนื้อหา (Text Content):** เปิด ChatGPT, Gemini, หรือ Claude บน Browser
2. **พิมพ์ Prompt บังคับโครงสร้าง Markdown:**
   *"ฉันชื่อ [ชื่อคุณ] เป็นนิสิตสาขา [ชื่อสาขา] อยากทำเว็บ Portfolio แนะนำตัว ช่วยเขียนสโลแกนเท่ๆ และแนะนำตัวแบบเป็นกันเอง พร้อมลิสต์ความสนใจ 3 ข้อ* 
   ***ข้อสำคัญ: โปรดจัดรูปแบบคำตอบทั้งหมดด้วย Markdown โดยใช้ `#` สำหรับชื่อเว็บ, `##` สำหรับหัวข้อย่อย, และ `-` สำหรับลิสต์ความสนใจ เพื่อให้ฉันเอาไปทำเว็บต่อได้ง่ายๆ"***
3. ก๊อปปี้เนื้อหาที่เป็น Markdown ที่ AI ตอบกลับมา เตรียมเอาไว้
4. **สร้างรูปภาพ (Image Content):** สั่งให้ AI ตัวเดิม (หรือเว็บอย่าง Copilot/Gemini) วาดรูปให้
   *"ช่วยวาดรูปอวาตาร์สไตล์การ์ตูน 3D มินิมอล โทนสีฟ้าพาสเทล เพื่อใช้เป็นรูปโปรไฟล์บนเว็บให้หน่อย"*
5. เซฟรูปภาพที่ได้มาไว้ที่ Desktop ตั้งชื่อว่า `profile.png`

### Step 2: เตรียมพื้นที่ทำงานและประกอบร่างแปลน (Workspace & Requirements)
1. เปิดโปรแกรม **Antigravity**
2. ใช้ช่องแชทสั่งให้ AI สร้างโฟลเดอร์ชื่อ `my-portfolio` ไว้ที่ Desktop
3. สั่งให้สร้างโฟลเดอร์ย่อยชื่อ `images` ไว้ข้างใน และลากรูป `profile.png` ไปใส่ไว้ในโฟลเดอร์นั้น
4. สั่งให้ AI สร้างไฟล์ชื่อ `requirements.md` ไว้ใน `my-portfolio`
5. เปิดไฟล์ `requirements.md` ขึ้นมา นำ Content ที่ได้จาก Step 1 มาวาง และเพิ่มส่วนของ "ดีไซน์" เข้าไปต่อท้าย
   > **ตัวอย่างเนื้อหาในไฟล์:** 
   > *(วางเนื้อหาที่ก๊อปปี้มาจาก AI ใน Step 1 ที่เป็น Markdown แล้ว)*
   >
   > **รูปภาพที่ต้องใช้:**
   > - รูปโปรไฟล์: `images/profile.png`
   > 
   > **Theme ดีไซน์:** อยากได้เว็บแนวมินิมอล ดูสะอาดตา ใช้ฟอนต์อ่านง่าย โทนสีพาสเทล

### Step 3: ให้ AI รับเหมาก่อสร้าง (Code Generation)
1. พิมพ์คำสั่งในช่องแชทของ Antigravity ว่า:
   *"ช่วยอ่านไฟล์ `requirements.md` แล้วสร้างไฟล์ `index.html` โดยเขียนโค้ด HTML/CSS ทั้งหมดรวมในไฟล์เดียวให้ตรงตาม Requirement ที"*
2. ดับเบิ้ลคลิกไฟล์ `index.html` เพื่อเปิดดูผลลัพธ์บน Browser (เช่น Chrome/Safari)

### Step 4: สั่งแก้ดีไซน์ด้วย "วิถีสถาปนิก" (Iterative Prompting)
1. ให้หาจุดที่คุณอยากปรับแก้บนหน้าเว็บ (อย่างน้อย 2 จุด)
2. **กฎเหล็ก:** ห้ามสั่งแก้โค้ดตรงๆ! ให้พิมพ์บอก AI ว่า *"อยากแก้... ช่วยอัปเดตไฟล์ `requirements.md` ให้หน่อย"*
3. เมื่อตรวจสอบไฟล์ `.md` แล้วว่าอัปเดตถูกต้อง ให้สั่ง AI ว่า *"ช่วยแก้ไฟล์ `index.html` ให้ตรงตาม requirement ล่าสุดที"*

### Step 5: เอาเว็บขึ้นออนไลน์ (Deployment)
1. สมัครและล็อคอินเข้าเว็บ **GitHub.com**
2. สร้าง Repository ใหม่ชื่อ `my-portfolio` (ตั้งเป็น Public)
3. กลับมาที่โปรแกรม Antigravity เปิด Terminal (หน้าจอดำ) แล้วพิมพ์คำสั่ง 3 ทหารเสือ:
   - `git add .`
   - `git commit -m "first commit"`
4. นำคำสั่ง `git branch...`, `git remote...` และ `git push...` จากเว็บ GitHub มาวางและกด Enter เพื่อดันโค้ดขึ้นเว็บ
5. ไปที่ Settings > Pages ใน GitHub แล้วเปิดใช้งาน **GitHub Pages** (เลือก branch `main`)
6. รอ 1 นาที คุณจะได้ลิงก์หน้าเว็บของคุณ (URL)

✅ **สิ่งที่ต้องส่งในคลาส:** 
- ก๊อปปี้ลิงก์ URL ของเว็บไซต์คุณ (เช่น `https://yourname.github.io/my-portfolio/`) ไปวางในระบบส่งงานของรายวิชา

## 🏠 Part 2: การบ้าน (Homework)
*ให้ทำต่อที่บ้าน และอัปเดตไฟล์ขึ้น GitHub ก่อนเริ่มคาบเรียนถัดไป*

### 1. พัฒนาเว็บไซต์ให้สมบูรณ์ขึ้น (Upgrade)
- ใช้หลักการ "วิถีสถาปนิก" อัปเดตไฟล์ `requirements.md` เพื่อเพิ่มฟีเจอร์ต่อไปนี้:
  1. เพิ่ม **"ตาราง (Table)"** แสดงข้อมูลตารางเรียน หรือ ประวัติการศึกษา
  2. เพิ่มปุ่ม **"ติดต่อฉัน"** ที่ลิงก์ไปหา Facebook หรือ Instagram ของคุณ
- สั่งให้ AI อัปเดตไฟล์ `index.html` ตาม Requirement ที่เพิ่มมา

### 2. เขียนสรุปการเรียนรู้ (Reflection)
- เปิดไฟล์ `requirements.md` เดิม
- พิมพ์เพิ่มหัวข้อใหม่ด้านล่างสุดชื่อว่า `## สรุปการเรียนรู้ Session 1`
- เขียนตอบคำถาม 2 ข้อนี้สั้นๆ (ใช้ Markdown ด้วย):
  1. *สิ่งที่ค้นพบว่าน่าทึ่งที่สุด หรือสนุกที่สุด ในการให้ AI เขียนเว็บให้คืออะไร?*
  2. *คุณคิดว่าการพิมพ์คำสั่ง Git อัปเดตงานขึ้น GitHub มีประโยชน์อย่างไรเมื่อเทียบกับการลากไฟล์ไปวางธรรมดา?*

### 3. อัปเดตงานขึ้น GitHub (Git Push)
เมื่อแก้โค้ดและเขียนสรุปเสร็จแล้ว ให้อัปเดตงานทั้งหมดขึ้นออนไลน์:
1. เปิด Terminal
2. พิมพ์ `git add .`
3. พิมพ์ `git commit -m "add table and reflection"`
4. พิมพ์ `git push`
5. เช็คความเรียบร้อยบน URL เดิมของคุณ! 

*(ไม่ต้องส่งลิงก์ใหม่ เพราะลิงก์เดิมบน GitHub จะอัปเดตอัตโนมัติ อาจารย์จะเข้าไปตรวจจากลิงก์เดิมที่คุณส่งไว้ในคาบ)*
