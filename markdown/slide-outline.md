# 🟦 **Session 1 — Build & Deploy Your First Webpage with AI**  
### **Slide Outline**
1. Title: Build & Deploy Your First Webpage with AI  
2. เป้าหมายของคาบนี้  
3. AI‑First Web Development คืออะไร  
4. เว็บประกอบด้วยอะไรบ้าง (HTML / CSS / JS แบบภาพรวม)  
5. **Markdown คืออะไร?** (`.md` ไฟล์ที่ AI ใช้ตลอดเวลา)  
6. **ทำไม Markdown ถึงสำคัญกับ AI** — AI อ่าน/เขียน Markdown ตลอด  
7. **Markdown Syntax พื้นฐาน** — `#` หัวข้อ, `-` ลิสต์, ` ``` ` code block  
8. **Workshop: เขียน Prompt แบบ Markdown** — ทดลองสั่ง AI จริง  
9. ตัวอย่าง Landing Page ที่ AI สร้างได้  
10. เปิดโค้ดที่ AI สร้าง (โครงสร้าง HTML)  
11. ปรับแต่งข้อความ/สี/รูปภาพด้วย AI  
12. การเตรียมไฟล์สำหรับ deploy  
13. ส่งโค้ดขึ้น GitHub ด้วยคำสั่ง Git (`git add`, `commit`, `push`)
14. เปิดใช้งาน GitHub Pages ให้เว็บออนไลน์
15. 🎬 Demo: ให้ AI ใน Editor ช่วยพิมพ์คำสั่ง Git แทนเรา (Automation)
16. แชร์ลิงก์เว็บของนิสิต  
17. สรุปสิ่งที่เรียน  
18. ใบงาน + การบ้าน
