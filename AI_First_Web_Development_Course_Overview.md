# AI-First Web Development Course Overview

> **แนวคิดหลัก:** สอนนิสิตให้ **"คุม AI"** มากกว่าสอนให้เขียนโค้ดเอง
> แต่ยังคง **"เข้าใจพื้นฐานเว็บ"** แบบที่จำเป็นต่อการสั่ง AI

นี่คือหลักสูตร AI-First Web Development ที่สมบูรณ์ที่สุดสำหรับผู้เริ่มต้น

---

## 📋 ภาพรวมหลักสูตร

| รายการ | รายละเอียด |
|---|---|
| **ระยะเวลา** | 6 คาบ × 2 ชั่วโมง 45 นาที |
| **กลุ่มเป้าหมาย** | นิสิตไม่มีพื้นฐานเขียนเว็บมาก่อน |
| **เป้าหมายหลัก** | สร้างเว็บ/เว็บแอพจริงได้ โดยใช้ AI เป็นผู้ช่วยหลัก |

---

## 📚 สรุป 6 คาบเรียน

1. **Build & Deploy Your First Webpage with AI**
2. **Mastering Web Layout by Working with AI**
3. **Interactive JavaScript & AI-Assisted Debugging**
4. **Building AI-Generated Web Apps with Real Logic**
5. **AI-Driven API Integration & Dynamic Data**
6. **Final Project: AI-Powered Web/App Development**

---

## 🚀 Session 1 — Build & Deploy Your First Webpage with AI

*สร้างเว็บแรก + Deploy ตั้งแต่วันแรก*

### 🎯 เป้าหมายคาบนี้
- ให้นิสิต "มีเว็บออนไลน์ของตัวเอง" ภายในคาบแรก
- สร้างความมั่นใจว่า AI สามารถช่วยสร้างเว็บได้จริง
- เข้าใจภาพรวมของเว็บแบบไม่ลงลึก

### 📖 เนื้อหา
- เว็บทำงานอย่างไร (HTML/CSS/JS แบบภาพรวม 15 นาที)
- **Markdown คืออะไร และทำไมสำคัญกับ AI**
  - AI อ่านและเขียน Markdown ตลอดเวลา
  - การเขียน Prompt แบบ Markdown ช่วยให้ AI เข้าใจได้ชัดขึ้น
  - ไฟล์ `.md` ใช้เป็น context / instruction ให้ AI ได้
- วิธีสั่ง AI ให้สร้าง Landing Page
- การแก้ไขข้อความ/รูปภาพ/สี ด้วยการสั่ง AI
- Deploy ขึ้นออนไลน์ผ่าน **GitHub Pages** (เพื่อให้แก้โค้ดอัปเดตง่ายในอนาคต)
- ให้ AI อธิบายโครงสร้างเว็บที่มันสร้าง
- 🎬 **Demo (ดูอย่างเดียว):** การใช้ AI ใน Editor ช่วย Commit และ Push ขึ้น GitHub อัตโนมัติ

### 🛠️ Workshop
- เขียน Markdown เบื้องต้น (หัวข้อ, ลิสต์, code block)
- ใช้ Markdown เป็น Prompt สั่ง AI สร้าง Landing Page
- ปรับแต่งด้วย AI (แก้ไข Requirement ก่อนเสมอ)
- Deploy ด้วย GitHub Pages และแชร์ลิงก์

### ✅ Learning Outcomes
- มีเว็บออนไลน์ของตัวเอง
- รู้วิธีสั่ง AI ให้สร้างหน้าเว็บ
- เข้าใจว่า Markdown ช่วยให้คุย AI ได้มีประสิทธิภาพขึ้น
- เข้าใจภาพรวมเว็บแบบไม่เครียด

## 🎨 Session 2 — Mastering Web Layout by Working with AI

*เข้าใจโครงสร้างเว็บ + ใช้ AI ออกแบบ layout*

### 🎯 เป้าหมายคาบนี้
- เข้าใจ layout ของเว็บแบบมืออาชีพ
- ใช้ AI ช่วยออกแบบและปรับ layout
- เข้าใจ HTML/CSS เท่าที่จำเป็นต่อการสั่ง AI

### 📖 เนื้อหา
- โครงสร้างเว็บ: section, container, grid, spacing
- HTML/CSS ที่จำเป็นจริง ๆ
- การใช้ Google Stitch เพื่อสร้างและปรับแต่ง UI Design ด้วย AI
- ให้ AI อธิบาย HTML/CSS (กล่อง, ระยะห่าง, สี)
- ให้ AI ปรับ layout ให้สวยขึ้น

### 🛠️ Workshop
- ให้ AI redesign หน้าเว็บของนิสิต
- ปรับ spacing, color, typography
- ทำ responsive layout ด้วย AI

### ✅ Learning Outcomes
- อ่าน HTML/CSS ออก
- สั่ง AI ให้ปรับ layout ได้
- ใช้เครื่องมือออกแบบ AI ได้

## ⚡ Session 3 — Interactive JavaScript & AI-Assisted Debugging

*ทำเว็บให้ขยับได้ + Debug แบบง่ายด้วย AI*

### 🎯 เป้าหมายคาบนี้
- เข้าใจ JavaScript แบบไม่ยาก
- ทำเว็บ interactive ได้
- ใช้ AI ช่วย debug

### 📖 เนื้อหา
- DOM คืออะไร (แบบง่าย)
- Event Listener
- การเปลี่ยนข้อความ/สี/คลาส
- Debugging ด้วย DevTools
- ให้ AI อธิบาย error และแก้บั๊ก

### 🛠️ Workshop
- Counter App
- Theme Switcher
- Popup Modal
- Debug โค้ดที่มีบั๊กง่าย ๆ

### ✅ Learning Outcomes
- ทำเว็บ interactive ได้
- เข้าใจ JS พอที่จะสั่ง AI
- Debug ด้วย AI ได้

## 🧠 Session 4 — Building AI-Generated Web Apps with Real Logic

*จากหน้าเว็บ → เว็บแอพที่มีตรรกะจริง*

### 🎯 เป้าหมายคาบนี้
- เข้าใจ "logic" ของเว็บแอพ
- ใช้ AI สร้างเว็บแอพง่าย ๆ
- ปรับฟีเจอร์ด้วยการสั่ง AI

### 📖 เนื้อหา
- State คืออะไร
- การอัปเดต UI ตามข้อมูล
- ให้ AI สร้าง Web App เช่น
  - To-Do App
  - Notes App
  - Calculator
- ให้ AI refactor โค้ดให้สั้นลง

### 🛠️ Workshop
- เลือกทำ 1 เว็บแอพ
- ปรับฟีเจอร์ด้วย AI
- เพิ่ม UI/UX ด้วย AI

### ✅ Learning Outcomes
- เข้าใจ logic ของเว็บแอพ
- ใช้ AI สร้างแอพได้
- ปรับฟีเจอร์ได้ด้วยการสั่ง AI

## 🌐 Session 5 — AI-Driven API Integration & Dynamic Data

*ดึงข้อมูลจริงจาก API โดยใช้ AI เป็นตัวหลัก*

### 🎯 เป้าหมายคาบนี้
- เข้าใจ API แบบไม่ต้องเขียนเอง
- ใช้ AI สร้างโค้ด fetch API

### 📖 เนื้อหา

- JSON คืออะไร
- ให้ AI สร้าง API App เช่น
  - Weather App
  - Random Quote
  - Image Search
- Debug API ด้วย AI

### 🛠️ Workshop
- เลือกทำ 1 API App
- ให้ AI ปรับ UI/UX
- ให้ AI อธิบาย error message

### ✅ Learning Outcomes
- เข้าใจ API
- ใช้ AI เขียน API integration ได้
- Debug API ได้ด้วย AI

- Deploy
- Present

- ทำโปรเจกต์จริง

### ✅ Learning Outcomes

- Deploy และ present ได้
